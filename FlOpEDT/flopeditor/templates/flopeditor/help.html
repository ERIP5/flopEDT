Documentation de FlopEditor

Flopeditor est une application développée dans le cadre de FlopEDT par quatre étudiants de M1 Informatique de l'Université Paul Sabatier. Son but est de permettre aux professeurs et autres intervenants de modifier tous les éléments liés à la gestion de leurs départements, de la création de promotions à l'attribution de salles, en passant par la gestion des responsables et tuteurs de leur département.

Table des matières récapitulant les droits

Dans FlopEDT, trois rôles peuvent être attribués à un utilisateur : Tuteur, Responsable d'un département et Superuser.
- Un tuteur peut visualiser toutes les données (excepté la liste des intervenants) mais ne peut pas les modifier.
- Un responsable d'un département peut visualiser toutes les données de tous les départements et peut modifier les données concernant son département.
- Un superuser a tous les droits. Il peut visualiser et modifier toutes les données de tous les départements de FlopEDT. Il peut également créer un département, ou modifier les attributs d'un département existant, comme les responsables de celui-ci.

Créer un département


Il est possible de créer un départemment à partir de la page d'accueil de Flopeditor. Pour cela, cliquez sur le bouton en bas à droite, ce qui ouvrira le menu de création d'un département. A partir de là, vous pourrez ajouter le nom, l'abréviation ainsi que les responsables du département à créer.




Modifier un département

Il est possible de modifier le nom, l'abrevviation et les responsables d'un départemment à partir de la page d'accueil de Flopeditor. Pour cela, à côté du bouton portant le nom du département, se trouve un bouton en forme de crayon. En cliquant sur celui-ci un menu s'ouvre, permettant de modifier les attributs du département. A partir de ce menu, vous pourrez modifier le nom, l'abréviation ainsi que les responsables du département à créer.



Ajouter, modifier, supprimer un intervenant

Note : Si vous n'êtes pas responsable d'un département, vous ne pouvez pas voir accéder à cette partie.

Le premier élément de la barre de navigation, en gaut à gauche, permet d'accéder au tableau des intervenants. A partir de ce tableau, vous pouvez créer un intervenant, l'ajouter ou le retirer d'un département, modifier des éléments le caractérisant, ou encore le supprimer de la liste.



Ajouter un intervenant

La première ligne vous permet d'ajouter un intervenant à la liste. Pour cela, il faut y renseigner son ID, son prénom, son nom de famille et son statut. Si l'intervenant à ajouter est vacataire, il faut également renseigner son employeur. Vous avez également la possibilité d'affecter à cet intervenant un ou plusieurs départements.

Si vous affectez un département à un intervenant, il n'en devient pas responsable. Pour ajouter un responsable à un département, reportez-vous à la section Création, modification d'un département.


Modifier un intervenant

Pour modifier un intervenant, vous pouvez cliquer sur le bouton avec le symbole de crayon, tout à droite de la ligne. Vous pouvez modifier son ID, son prénom, son nom de famille et son statut. Si l'intervenant à modifier est vacataire, il faut également renseigner son employeur. Vous avez également la possibilité d'affecter ou de retirer un ou plusieurs départements à cet intervenant.



Supprimer un intervenant

Si vous voulez supprimer un intervenant, il vous suffit d'appuyer sur le bouton rouge tout à droite de la ligne. Un avertissement apparaît pour vous demander de confirmer la suppression. Il ne vous reste qu'à valider la suppression.

Attention : Si vous supprimez cet intervenant, tous les cours associés seront supprimés et ce dernier ne pourra plus se connecter.

L'onglet "Salles" vous permet d'accéder à la liste des salles enregistrées dans FlopEDT. A partir de ce tableau, vous pouvez notamment créer une salle ou un groupe de salles, ou encore modifier ou supprimer une salle existante. Ajouter une salle

La première ligne vous permet d'ajouter une salle à la liste. Pour cela, vous devez au moins renseigner le nom de la salle. Si vous le souhaitez, vous pouvez l'associer à un département ou à un groupe de salles.

Si vous souhaitez créer un groupe de salles, faites comme si c'était une salle simple : Indiquez un nom, puis appuyez sur le boton "+", et enregistrez les modifications. Après avoir actualisé la page, votre nouveau groupe de salles apparaitra bien dans la liste "Sous-salle de..." !


Modifier une salle

Pour modifier une salle, vous pouvez cliquer sur le bouton avec le symbole de crayon, tout à droite de la ligne. Vous pouvez modifier son nom, ses sous-salles associées et les départements auxquels la salle est affectée.



Supprimer une salle

Si vous voulez supprimer une salle, il vous suffit d'appuyer sur le bouton rouge tout à droite de la ligne. Un avertissement apparaît pour vous demander de confirmer la suppression. Il ne vous reste qu'à valider la suppression.

Note : Si vous supprimez une salle comportant des sous-salles, les sous-salles ne sont pas supprimées.

Note : Si vous supprimez une catégorie de salle, les salles et groupes de salles associées ne sont pas supprimées.


Ajouter, Modifier, supprimer un semestre

Note : Si vous n'êtes pas responsable d'un département, vous pourrez uniquement visualiser la liste des semestres.

Pour atteindre le tableau permettant de visualiser et modifier les semestres : vous devez vous rendre dans l'onglet "Module" dans la barre de navigation en haut à droite puis dans l'onglet "SEMESTRE" au-dessus du bouton "ENREGISTRER".



Ajouter un semestre

  1. Sur la première ligne : renseigner l'id du semestre (exemples : S1, S2 ou S3; il doit être unique), le numéro de la semaine de début entre [1-53] et le numéro de la semaine de fin du nouveau semestre entre [1-53]
  2. Appuyez ensuite sur le bouton "+" à droite
  3. Répétez les 2 premiers points si vous voulez ajouter d'autre semestre
  4. Appuyez sur le bouton "ENREGISTRER" à gauche si vous n'avez plus de semestre à ajouter.

Modifier un semestre

  1. Trouvez le semestre à modifier dans le tableau, puis cliquer sur l'icône de modification (crayon bleu) à droite
  2. Modifiez les champs que vous souhaitez (Id doit être unique, le numéro des semaines comprit entre [1-53]) puis cliquer sur à droite
  3. Répétez les 2 premiers points si vous voulez modifier d'autre semestre
  4. Cliquez sur "ENREGISTRER" en haut à gauche si vous n'avez plus de semestre à modifier.



Supprimer un semestre

  1. Trouvez le semestre à supprimer dans le tableau, puis cliquer sur l'icône de suppression ( rouge) à droite
  2. Cliquez sur "OUI" si vous êtes sûr de vouloir supprimer ce semestre, "NON" sinon.
  3. Répétez les 2 premiers points si vous voulez supprimer d'autre semestre
  4. Cliquez sur "ENREGISTRER" en haut à gauche si vous n'avez plus de semestre à supprimer.

Attention : Si vous supprimez un semestre, tous les modules associés seront supprimés.

Ajouter, Modifier, supprimer une nature de groupes

Note : Si vous n'êtes pas responsable du département, vous pourrez uniquement visualiser la liste des natures de groupes.

Pour atteindre le tableau permettant de visualiser et modifier les natures de groupes (TP, TD, CE...) : vous devez vous rendre dans l'onglet "élève" dans la barre de navigation en haut à droite puis dans l'onglet "SEMESTRE" au-dessus du bouton "ENREGISTRER".



Ajouter une nature de groupes

  1. Sur la première ligne : renseigner l'id de la nature de groupes (exemples : TD ou TP; il doit être unique et < 50 caractères)
  2. Appuyez ensuite sur le bouton "+" à droite
  3. Répétez les 2 premiers points si vous voulez ajouter d'autre nature de groupes
  4. Appuyez sur le bouton "ENREGISTRER" à gauche si vous n'avez plus de nature de groupes à ajouter.

Modifier une nature de groupes

  1. Trouvez la nature de groupes à modifier dans le tableau, puis cliquer sur l'icône de modification (crayon bleu) à droite
  2. Modifiez le champ Id (doit être unique et < 50 caractères) puis cliquez sur le bouton à droite pour valider.
  3. Répétez les 2 premiers points si vous voulez modifier d'autre nature de groupes
  4. Cliquez sur "ENREGISTRER" en haut à gauche si vous n'avez plus de nature de groupes à modifier.



Supprimer une nature de groupes

  1. Trouvez la nature de groupes à supprimer dans le tableau, puis cliquer sur l'icône de suppression à droite
  2. Cliquez sur "OUI" si vous êtes sûr de vouloir supprimer cete nature de groupes, "NON" sinon.
  3. Répétez les 2 premiers points si vous voulez supprimer d'autre nature de groupes
  4. Cliquez sur "ENREGISTRER" en haut à gauche si vous n'avez plus de nature de groupes à supprimer.

ATTENTION : Si vous supprimez une nature de groupes, tous les groupes d'élèves et les cours associés seront supprimés.

Ajouter, Modifier, supprimer un module

A écrire !

Note : Si vous n'êtes pas responsable du département, vous pourrez uniquement visualiser la liste des modules.
L'onglet "Modules" de la barre de navigation vous permet d'accéder à la liste des modules enregistrés dans FlopEDT.

Ajouter un module

  1. Dans la première ligne du tableau, saisissez les informations du module que vous souhaitez ajouter. Il est obligatoire de saisir une abbréviation, un code PPN et un nom pour le module.
  2. Appuyez sur le bouton '+' à droite de la ligne.
  3. Répétez les étapes 1 et 2 si vous souhaitez ajouter d'autres modules.
  4. Appuyez sur le bouton 'Enregistrer' pour valider les modifications.
Note : pour ajouter un module, il est nécessaire d'avoir ajouté au moins une promo, un intervenant et un semestre au département.


Modifier un module

  1. Après avoir choisi un module à modifier, appuyez sur le bouton bleu à droite de la ligne correspondante.
  2. Vous pouvez alors modifier l'abbréviation, le code PPN, le nom, la description, la promo, l'enseignant responsable et le semsestre du module.
  3. Appuyez sur le bouton vert à droite de la ligne pour valider l'édition du module.
  4. Répétez les étapes 1, 2 et 3 si vous souhaitez modifier d'autres modules.
  5. Appuyez sur le bouton 'Enregistrer' pour valider les modifications.


Supprimer un module

  1. Après avoir choisi un module à supprimer, appuyez sur le bouton rouge à droite de la ligne correspondante.
  2. Confirmez la suppression en cliquant sur 'oui'.
  3. Répétez les étapes 1 et 2 si vous souhaitez supprimer d'autres modules.
  4. Appuyez sur le bouton 'Enregistrer' pour valider les modifications.

Afficher les paramètres

  1. Depuis une page d'un département de flopeditor, cliquez sur le bouton 'Paramètres' à droite de la barre de navigation.
  2. Les paramètres de ce département s'affichent.


Modifier les paramètres

  1. Sur la page des paramètres d'un départment, cliquez sur le bouton 'Modifier' en bas.
  2. Vous pouvez modifier les jours ouvrables et les horaires par défaut du département.
  3. Une fois que toutes les modifications ont été apportées, appuyez sur le bouton 'Valider' en bas à gauche.
  4. Quand les modifications sont validées, les champs d'édition disparaissent.
Note : L'heure de début des cours doit être inférieure à l'heure de fin des cours, les heures du déjeuner doivent être comprises entre les heures de début et de fin des cours.

Afficher votre profil

  1. Depuis une page de flopeditor cliquez sur le bouton en forme de bonhomme en haut à droite.

  4. Quand les modifications sont validées, la fenêtre du profil disparaît.
